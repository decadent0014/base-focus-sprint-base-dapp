# Base Focus Sprint

## App Name

Base Focus Sprint

## Tagline

Start one sprint

## Description

Log one focused work sprint on Base with a task, duration, and short note so your session has a visible start point.

Character count: 113

## Category

Focus / timer / onchain utility

## Short Pitch

Base Focus Sprint gives people a clear way to stamp one focused work session on Base with a task, duration, and short note.

## Production Environment Variables

- `NEXT_PUBLIC_FOCUS_SPRINT_CONTRACT_ADDRESS`: deployed focus sprint contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
