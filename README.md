# Base Focus Sprint

Dear reviewer,

This is a small Base project about focus sprints. It lets a visitor starting a timed work push and leaves a sprint receipt behind. The project is not trying to prove ownership through inboxes; it proves it with the same public pieces a user can open and verify.

The Base Dashboard registration is present under Build ID `6a0587b48f636ba200aa0243`. The public builder wallet is `0x9115Af9a084d951688d4291d82e08336131da702`. Transaction attribution uses Builder Code `bc_35hvyvha`. The live deployment is https://base-focus-sprint.vercel.app, and the source repository is https://github.com/decadent0014/base-focus-sprint-base-dapp.

If you want to run it locally:

```bash
npm install
npm run dev
```

The working parts are simple: Next.js, React, TypeScript, Tailwind CSS, wagmi, viem, Base, Vercel. Local secrets stay local; `.env.example` is the only environment file meant for Git.
